# Getting started

```bash
git clone https://github.com/thenaim/ion-tk-social-network myApp
cd myApp
npm install
npm start
```

# Features

- Lazy-loading
- Auth pages (login, register, forgot and reset)
- Themes (default and dark)
- Ngrx
- LocalStorage
- i18n (English and Russion)
- Responsive design
